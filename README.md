# FlowerApp

A responsive, single-page website that helps people discover the perfect flowers for any occasion.

## Project Overview

**FloralXpress** is a flower showcase website designed to guide visitors in selecting appropriate blooms for different life moments—whether it's a romantic date, a wedding celebration, a sympathy gesture, or just because. The site features curated flower selections for each occasion with humorous, approachable copy that makes flower gifting feel fun and intuitive.

## How It's Built

- **HTML Structure**: Clean, semantic HTML with a single-page layout featuring a sticky navigation menu, hero banner, and five main sections (Dates, Weddings, Sympathy, Gifts, and About)
- **Responsive Design**: Built with CSS Grid and Flexbox for flexible layouts, featuring mobile-first media queries (breakpoints at 768px and 480px)
- **Interactive Features**: Hamburger menu for mobile navigation with smooth animations
- **Styling**: External CSS file with modular, well-organized sections for easy maintenance
- **Pure Frontend**: No backend dependencies—lightweight and deployable anywhere

## Key Features

 Responsive navigation that adapts to all screen sizes  
 Beautiful card-based layout showcasing flower arrangements  
 Mobile-optimized design for perfect viewing on any device  
 Smooth animations and hover effects for better UX  
 Occasion-based flower recommendations with personality
